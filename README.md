# Graficador de Vectores

Este es un programa en Python que permite graficar vectores en un plano 2D. Utiliza **Tkinter** para la interfaz gráfica y **Matplotlib** para mostrar el gráfico.

## Requisitos

Necesitarás tener instaladas las siguientes bibliotecas:

- Tkinter
- Matplotlib

Puedes instalar las dependencias con:

```bash
pip install matplotlib tk
```

## Uso

1. Clona o descarga el repositorio.
2. Ejecuta el archivo `vector_plotter.py`:

```bash
python vector_plotter.py
```

3. Ingresa las componentes del vector en los campos de texto y haz clic en "Graficar Vector".
